Full-Text Search Demystified and Awesome Logging Infrastructure Using The Elastic Stack

2 December 2016, Riga, Latvia

Course description

Full-Text Search Demystified

Today’s applications are expected to provide powerful full-text search. But how does that work in general and how do I implement it on my site or in my application? Actually, this is not as hard as it sounds at first. This workshop covers:

How full-text search works in general and what the differences to databases are

How to implement common requirements with Elasticsearch

Attendees will learn how to add common search patterns to their applications without breaking a sweat.

Awesome Logging Infrastructure Using The Elastic Stack

One of the hottest and most widely used tools for log monitoring and analytics at the moment is the Elastic Stack:

Elasticsearch doing the hard work of analyzing and searching large amounts of data

Logstash and Beats for collecting them

Kibana for powerful visualizations

This worshop gives you an overview of the four technologies, how they are working together, and how they can help you solve your problems.

This course offers concrete practices that you can apply at work immediately.
