Building Cloud Native microservices with Spring Cloud & Netflix OSS

2-day in-house training

Course description

Do you want to build an application using microservices and don't know where to start? How do services find each other? How do I configure many instances of each service? How do I see what's going on with my services? If you've ever asked any of these questions, this training will give you some answers using "Spring Cloud" and "Netflix OSS" to create microservices that are robust, discoverable, scalable and configurable.

Learn core concepts about Spring Cloud and Netflix OSS and how they are integrated. See demos of and learn how to use the following:

Spring Cloud Config Server (distributed configuration)

Eureka (service registration and discovery)

Hystrix (circuit breaker/fault tolerance library)

Hystrix Dashboard (service health dashboard)

Ribbon (client side load balancing, including RestTemplate integration)

Feign (easy rest clients)

Zuul (routing)

Spring Cloud Bus (distributed Spring Boot actuator)

Distributed tracing with Spring Cloud Sleuth

...and much more!

This course offersconcrete practicesthat you can apply at work immediately.
